API calls list

Url Methods Description Parameters
/api/forecast GET, PUT, DELETE Manages forecast data: supports retrieving, uploading or deleting forecast data
  • key - API key (required)
  • placename - Place name (optional)
  • lat, lon - Coordinates (optional)
  • coords - Coordinates (optional)
  • placeid - Place ID (optional)
  • date - Date in ISO format (optional)
  • details - Include additional details (optional)
/api/now GET, PUT Manages current weather data: supports retrieving and uploading current weather data
  • placename - Place name (optional)
  • lat, lon - Coordinates (optional)
  • coords - Coordinates (optional)
  • placeid - Place ID (optional)
/api/places GET Retrieves place data: supports querying by name or coordinates
  • placename - Place name (optional)
  • lat, lon - Coordinates (optional)
  • coords - Coordinates (optional)
/api/conditions GET Retrieves weather condition data: supports querying by condition ID
  • id - Condition ID (optional)
/api/alltables GET Retrieves all data from all tables in the database
