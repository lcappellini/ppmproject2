API calls list

Url Methods Description Parameters
/api/forecast GET, PUT, DELETE Manages forecast data: supports retrieving, uploading or deleting forecast data
  • key - API key (required)
    GET
  • placename - Place name
  • placeid - Place ID
  • date - Date in ISO format (required)
  • details - Whether to include additional details (condition description and place name) (optional)
    PUT
  • placeid - Place ID
  • date - Date in ISO format (required)
  • condition - Weather condition id (optional)
  • rain - Rain (mm) (optional)
  • humidity - Humidity percentage (optional)
  • wind - Wind speed (km/h) (optional)
  • wind_direction - Wind direction (optional)
    DELETE
  • forecastid - Forecast id
/api/places GET, PUT, DELETE Retrieves place data: supports retrieving, uploading or deleting places data
  • key - API key (required)
    GET
  • placeid - Place id
  • placename - Place name
    PUT
  • placeid - Place id
  • placename - Place name
  • lat - Latitude (optional)
  • lon - Longitude (optional)
  • coords - Coordinates 'float,float' (optional)
    DELETE
  • placeid - Place id
  • placename - Place name
/api/conditions GET Retrieves weather condition data: supports querying by condition ID
  • key - API key (required)
    GET
  • id - Condition ID (optional)
/api/alltables GET Retrieves all data from all tables in the database (for debug purpose, since this project is a "demo")

Examples (GET requests)

  • .../api/forecast?placeid=1&date=2024-06-21&details=true&key=YOUR_API_KEY
  • .../api/forecast?placename=Prato&date=2024-06-20&key=YOUR_API_KEY
  • .../api/conditions?id=1&key=YOUR_API_KEY
  • .../api/places?placename=Firenze&key=YOUR_API_KEY
